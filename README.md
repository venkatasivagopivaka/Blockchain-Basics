# Blockchain Basics 🏗️

This repository contains my daily technical notes and synthesis of the **Blockchain Basics** curriculum provided by Cyfrin Updraft. 

It serves as the theoretical foundation for my journey into smart contract auditing. Rather than copying transcripts, these files represent my own synthesis of core concepts, consensus mechanisms, and cryptography, documented daily to build a public proof-of-work.

## Structure
Each markdown file corresponds to a specific lesson or core concept and is broken down into:
- **Core Concept:** The primary mechanism and the problem it solves.
- **Key Mechanisms:** How it works under the hood.
- **Security Perspective:** Why an auditor cares about this specific feature.
- **Terminology:** Quick reference glossary.
